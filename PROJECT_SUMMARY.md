## 🎬 CINEMATIC DESIGN ELEMENTS

### Film Grain
SVG-based noise overlay for authentic film texture

### Typography Hierarchy
```
H1: 80-144px (Hero headlines)
H2: 48-112px (Section headlines)
H3: 32-48px (Subsections)
Body: 16-24px (Content)
```

### Color Usage
```
Ink Black (#0B0B0B)    - Primary background
Pitch Black (#111111)  - Secondary background
Grass Green (#0F2A25)  - Accent background
Gold (#C6A85B)         - CTAs, accents, highlights
Chalk White (#EDEDED)  - Primary text
Warm Gray (#8A8478)    - Secondary text
```

### Motion Design
- Easing: cubic-bezier(0.4, 0, 0.2, 1)
- Duration: 300-800ms
- Delays: Staggered 100-200ms
- Hover: Scale 1.05, Y: -2px
- Parallax: 50% scroll transform
